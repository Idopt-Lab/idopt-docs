## Overview

The Regional Air Mobility (RAM) Design Operations project provides tools and frameworks for designing, simulating, and evaluating regional air mobility systems.

## What's Included

- **Life Cycle Cost (LCC) Analysis** — models and computes the full lifecycle cost of RAM vehicles and operations
- **Test Suite** — automated tests validating core computation logic
- **Examples** — reference scripts demonstrating key workflows

## Getting Started

### Prerequisites

- Python 3.10+
- Dependencies: `numpy`, `pandas`, `scipy`, `matplotlib`, `openpyxl`

### Install dependencies

```bash
pip install numpy pandas scipy matplotlib openpyxl
```

### Run tests

```bash
python -m unittest discover -s tests -p "test*.py"
```
